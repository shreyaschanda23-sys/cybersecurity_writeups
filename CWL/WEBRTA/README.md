# CyberWarfare Labs – Web Red Team Analyst (WebRTA) Labs

## Overview

This repository documents my hands-on experience completing the **CyberWarfare Labs Web Red Team Analyst (WebRTA)** training. The labs simulate real-world web application penetration testing scenarios, allowing me to practice reconnaissance, vulnerability assessment, exploitation, and post-exploitation analysis in an authorized environment.

The purpose of this repository is to showcase my penetration testing methodology, technical skills, and the security concepts learned throughout the training.

---

## Skills Demonstrated

- Web Application Penetration Testing
- Reconnaissance & Enumeration
- HTTP Request Analysis
- SQL Injection (SQLi)
- XML External Entity (XXE)
- Server-Side Request Forgery (SSRF)
- Insecure Direct Object Reference (IDOR)
- JWT Analysis
- Authentication & Authorization Testing
- OTP Security Assessment
- Burp Suite & Burp Intruder
- Data Encoding & Decoding
- Vulnerability Chaining

---

## Tools Used

| Tool | Purpose |
|------|---------|
| Burp Suite | HTTP interception and manual testing |
| Burp Intruder | Automated request testing |
| JWT Editor | JWT inspection and modification |
| Nmap | Network reconnaissance |
| Feroxbuster | Directory and endpoint enumeration |
